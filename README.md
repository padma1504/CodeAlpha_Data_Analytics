# Netflix Exploratory Data Analysis

## Objective
To analyze Netflix content and identify trends, patterns, and insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Kaggle

## Analysis Performed
- Data Exploration
- Missing Value Analysis
- Duplicate Check
- Statistical Summary
- Country Analysis
- Genre Analysis
- Content Growth Analysis

## Key Findings
- Movies dominate Netflix content.
- Director column contains the highest number of missing values.
- United States contributes the highest number of titles.
- Netflix content increased significantly after 2016.
