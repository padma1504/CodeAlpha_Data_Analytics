# Netflix Exploratory Data Analysis

## Objective
Perform exploratory data analysis on the Netflix dataset to understand data distribution, missing values, and trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
1. Dataset exploration
2. Missing value analysis
3. Content type distribution
4. Release year analysis
5. Rating analysis
6. Country analysis

## Conclusion
EDA helped identify important patterns and trends in the Netflix dataset.
