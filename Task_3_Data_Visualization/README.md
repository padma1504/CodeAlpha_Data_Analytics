Netflix Data Visualization

Project Overview

This project focuses on visualizing Netflix content data to identify trends, patterns, and insights using various graphical representations.

Objective

To transform raw Netflix data into meaningful visualizations that help understand content distribution, growth trends, ratings, genres, and country-wise contributions.

Dataset

Netflix Dataset containing information about:

- Movies and TV Shows
- Directors and Cast
- Countries
- Release Years
- Ratings
- Genres

Tools and Libraries Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Kaggle Notebook
- GitHub

Visualizations Performed

1. Movies vs TV Shows Count Plot
2. Top 10 Countries by Content Count
3. Content Added Over Years
4. Ratings Distribution
5. Top 10 Genres Analysis
6. Movies vs TV Shows Percentage Pie Chart

Key Insights

- Movies dominate Netflix's content library.
- The United States contributes the highest number of titles.
- Netflix experienced rapid content growth after 2016.
- Certain ratings are more common than others.
- Popular genres account for a significant portion of the catalog.
- Data visualization makes trends and patterns easier to understand.

Conclusion

Data visualization is an effective technique for exploring datasets and communicating insights. Through various charts and graphs, important trends within Netflix's content library were identified and analyzed successfully.
